## v0.16 — 2026-05-24

**Added**
- **モバイルのフロート追従プレビュー**：下にスクロールして変換後サンプルが画面外へ出ると、右上に小さく固定表示してスクロールに追従。タップで先頭へ戻る。本プレビューと内容は同期。
- フロートは**指でドラッグして移動**＋**2本指ピンチで拡大縮小**でき、位置・サイズを記憶。
- フロートの既定サイズは150px。モチーフの位置調整はトップの本プレビューでのみ行う（フロートは閲覧用）。
- PCでは表示されない（縦スクロールが出ない設計のため不要）。

**Changed**
- フッターを共通仕様に合わせ、**作者名「キノメノ」**を紹介URLとともに明示。

**Fixed**
- デスクトップの中途半端なウィンドウサイズで、こまかい設定を開くと下部がフッターと重なる崩れを修正（収まらない場合はメイン領域内でスクロール）。

## v0.11 — 2026-05-24

## v0.10 — 2026-05-24（バージョン管理の起点）

「ととのえ」Web版を、公開できる完成度に到達。デスクトップ版（高機能版）と合わせて初のバージョン番号を付与。

### Web版「ととのえ」
**Added**
- 画像のドラッグ＆ドロップ／クリック選択（1枚特化）
- モチーフ自動検出 → 中央配置 → note 1920×1006 へ自動変換
- リアルタイムプレビュー（設定変更で即時反映）
- **サイズスライダー**（モチーフの大きさ 50–160%）＝無料の主役機能、プレビュー直下に大きく配置
- 右下フローティングのダウンロードボタン（常時表示・画像未投入時はグレー）
- リセット（最初からやり直す／加工中の画像を破棄）
- 「追加要素」パスワード解放で **こまかい設定** を解禁
  - 出力サイズの複数選択（note / X / Instagram / YouTube / カスタム）
  - 出力形式（PNG / JPG / WebP）
  - 背景色（白・クリーム・グレー・濃灰・黒・自由色）
  - 複数サイズ選択時は ZIP 一括ダウンロード
- こまかい設定は折り畳み式・**開閉状態を保存**
- レスポンシブ（PC＝横並び／モバイル＝縦積み）
- すべてブラウザ内で処理（画像はサーバーに送信されない）
- テスト用クエリ `?lock=1`（パスワード前のロック状態を表示し、記憶も解除）

**Changed**
- デザインをミニマル（白基調＋くすみセージ `#5f8276` の差し色、明朝ロゴ）に刷新
- ヘッダーを細く・タイトルを小さく
- プレビュー周りのグレー余白を廃止し、出力範囲をキャンバスの枠線で明示
- 解放済み表示を大きなバッジから最小の「✓」マークへ
- 「設定変更で即反映」表示を削除（当然の挙動のため）
- 出力サイズ／形式／背景色を「こまかい設定」に集約
- 全体の文字コントラストを引き上げ（特にフッターの注記）

**Fixed**
- オリジナル画像のサムネイルが表示されない不具合（Blob URL を使用前に解放していた）を修正

### デスクトップ版（高機能版）
**Added**
- リアルタイムプレビュー
- フィルター（セピア／モノクロ／彩度／明度／コントラスト／ぼかし）
- 複数サイズ同時生成、テキスト追加（位置・サイズ・自動シャドウ）
- 背景色カスタマイズ、出力形式（PNG / JPG / WebP）
- バッチ処理（複数フォルダ）＋スケジュール実行
- プリセット保存／読込（JSON）
- フラットデザインUI（タブ構成：メイン変換／バッチ）

### バージョニング方針
- `0.x` 系：公開前〜初期。後方互換を気にせず素早く改善。
- メジャー機能追加で `0.minor` を進める（例：テキスト追加機能の搭載で v0.11 など）。
- 不具合修正のみの場合は必要に応じて `0.minor.patch`。
