# 詳細説明書（ヘビーユーザー向け）— ととのえ

note 用の画像を **1920×1006px** にきれいに整える画像ツール。
Web版（インストール不要・ブラウザ完結）とデスクトップ版（高機能）の2系統があります。

---

## 1. Web版「ととのえ」

### 1-1. 基本の流れ
1. 画像を**ドラッグ＆ドロップ**、または枠をクリックして選択（**1枚**）
2. 自動でモチーフを検出し、note サイズ（1920×1006）の中央に配置したプレビューが出る
3. **サイズ**スライダーでモチーフの大きさを微調整（50〜160%）
4. 右下の**↓ボタン**でダウンロード
5. やり直すときは最下部の**リセット**

> 対応形式：JPG / PNG / GIF / WebP / BMP（ブラウザが表示できる画像）
> 画像は**すべてブラウザ内で処理**され、サーバーには送信されません。

### 1-2. モチーフ自動検出の仕組み（ざっくり）
- 画像を1/3に縮小し、「白に近くないピクセル（しきい値240）」の範囲を検出します。
- その範囲の周囲に余白（約60px）を足して切り出し、キャンバス中央へ配置します。
- **白背景に被写体／イラストが乗った画像**で最もきれいに決まります。
- 真っ白などモチーフが取れない画像は、画像全体をそのまま配置します。

### 1-3. サイズスライダー（主役機能・無料）
- モチーフの大きさを 50〜160% で調整。プレビューに即反映。
- 「余白を広めに上品に」「思い切り大きく」など、好みの収まりに。

### 1-3b. テキスト追加（v0.11〜）
- サイズスライダー直下の「**テキスト**」欄に文字を入力すると、画像に1行のテキストが入ります。
- **無料**：位置は下・色は自動（背景に合わせて白／濃色、縁取りで視認性確保）。
- **解放後（こまかい設定）**：位置（上/中/下）、文字サイズ、文字色（自動＋白／濃色／自由色）、フォント（ゴシック／明朝）を変更可能。
- 空欄なら何も描画されません（通常の変換と同じ）。

### 1-4. 追加要素（パスワード解放）
右上の「**追加要素 ＋**」からコードを入力すると、**こまかい設定**が解禁されます。
- **解放コードはnoteの該当記事内に記載**（お持ちでない方は記事をご確認ください）。
- 一度解放すると、その端末（ブラウザ）に記憶され、次回から再入力不要。
- 解放後はヘッダー右に小さな「✓」が表示されます。

#### こまかい設定の内容
| 項目 | 内容 |
|---|---|
| 出力サイズ | note / X(1024×512) / Instagram(1080) / YouTube(1280×720) / **＋追加サイズ**（任意の幅×高さ） 複数選択可 |
| 出力形式 | PNG / JPG / WebP |
| 背景色 | 白・クリーム・グレー・濃灰・黒・**自由色**（カラーピッカー） |

- 複数サイズを選ぶと、**ZIPで一括ダウンロード**されます（ファイル名は `元名_サイズ.拡張子`）。
- こまかい設定は**折り畳み式**。開閉状態は記憶されます。

### 1-5. レスポンシブ
- **PC**：左にオリジナル（小）、右に変換後（大）。画面内に収まり縦スクロールは出ません。
- **スマホ**：縦積み（プレビュー→オリジナル→設定）。横スクロールは出ません。

### 1-6. テスト用クエリ
- `…/index.html?lock=1` を開くと、**パスワード前（ロック）状態**で表示され、解放の記憶も消えます。
  動作確認や、解放フローを最初から試したいときに使用します。

### 1-6b. 言語・クレジット・支援（v0.12〜）
- **言語**：開くと閲覧者の言語を自動判定（日本語／英語）。フッター右の「EN／日本語」でいつでも切替（選択は記憶）。
- **クレジット**：画面下部に作者「キノメノ」と紹介記事リンク。
- **支援**：Ko-fi（https://ko-fi.com/kinomeno）と note のチップ機能から応援できます。

### 1-7. よくある質問
- **Q. 解放コードを変えたい** → 開発者向けの `UNLOCK_HASH`（SHA-256）を差し替えます（仕様書参照）。
- **Q. 解放を取り消したい** → ブラウザのサイトデータ（localStorage）を消すか、`?lock=1` で開きます。
- **Q. オフラインで使える？** → 単一サイズのDLは可。**複数サイズのZIPだけ**は外部ライブラリ(JSZip)をネット取得します。

## 2. デスクトップ版（高機能版）

Windows 向け。フォルダ単位の大量処理や、テキスト・フィルターまで行いたい人向け。

### 2-1. 主な機能
- ドラッグ＆ドロップ／フォルダ指定
- リアルタイムプレビュー
- **フィルター**：セピア／モノクロ／彩度／明度／コントラスト／ぼかし
- **複数サイズ同時生成**（サイズごとにサブフォルダ出力）
- **テキスト追加**：内容・サイズ・位置（上/中/下）・自動シャドウ・日本語フォント対応
- 背景色、出力形式（PNG/JPG/WebP）
- **バッチ処理**：複数フォルダのジョブを登録して一括実行
- **スケジュール実行**：毎日指定時刻に自動でバッチ実行
- **プリセット**：設定一式を保存／呼び出し（`~/.note_image_generator/` に保存）

### 2-2. 使い方
1. `note画像生成アプリ.exe` を起動（`desktop/dist/` 内）
2. ソースフォルダと出力フォルダを指定
3. サイズ・形式・背景色・フィルター・テキストを設定
4. 「▶ 変換開始」
5. バッチ／スケジュールは「バッチ / スケジュール」タブから

### 2-3. ビルド（開発者向け）
```bash
cd desktop
python build_exe.py      # PyInstaller で EXE を生成 → desktop/dist/
```
必要環境：Python 3.10+ / Pillow 10+ / numpy 1.24+ / tkinterdnd2 0.3+

---

## 3. 出力ファイル名の規則
- 単一サイズ：`<元のファイル名>_<サイズ名>.<拡張子>`（例：`flower_note.png`）
- 複数サイズ（Web）：上記をまとめた `<元のファイル名>_ととのえ.zip`
- デスクトップの複数サイズ：出力フォルダ内に**サイズ名のサブフォルダ**を作成

---

## 4. プライバシー
- Web版：画像処理はブラウザ内で完結。アップロード・保存は行いません。
- 記憶するのは「解放状態」「こまかい設定の開閉」のみ（localStorage）。
